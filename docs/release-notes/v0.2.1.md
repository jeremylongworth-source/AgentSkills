# v0.2.1 Release Notes

This release moves AgentSkills from prerelease distribution status to a normal
release posture while keeping the expanded role-based bundles correctly labeled
as alpha workflows.

Recommended GitHub release state: normal release

Reason: v0.2.1 does not claim that every new workflow bundle is proven. It
stabilizes the public distribution path: install behavior, release validation,
docs validation, fresh-home smoke tests, and cross-platform GitHub Actions
release gates.

## What Changed

- Added `docs/stable-release-plan.md` to separate release stability from bundle
  maturity.
- Expanded GitHub Actions validation from Windows-only to a Windows, macOS, and
  Linux matrix.
- Updated release-gate scripts to use whichever PowerShell host is available,
  instead of assuming Windows PowerShell.
- Updated release process, release owner checklist, release handoff, changelog,
  README, and validation tracker for normal-release readiness.

## Stable Distribution Claim

The distribution and validation path is stable when:

- `scripts/release-check.ps1` passes locally with publish dry-run enabled.
- GitHub Actions release gates pass on Windows, macOS, and Linux.
- Published-tag install smoke passes for at least one representative skill.
- README, changelog, release notes, validation tracker, and bundle counts match
  the repository.
- Secret and local-path scans pass.

## Bundle Maturity Boundary

The expanded role-based bundles remain alpha unless separately promoted.

Scenario validation proves routing and structure. It does not prove real-world
adoption, legal/compliance readiness, financial correctness, or domain-specific
business outcomes.

Before promoting bundles beyond alpha, collect:

- real or anonymized creator and sponsor packets
- real application API contract packets
- third-party scripted skill audit packets
- source-grounded data and finance packets
- leadership-review packets for executive bundles
- reviewer packets for owner, operating, product, revenue, engineering, and
  support bundles
