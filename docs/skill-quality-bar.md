# Skill Quality Bar

A skill is ready when it meets these checks:

- triggers precisely from frontmatter description
- has a useful workflow another agent can follow
- includes a deliverable shape
- has at least one reference checklist
- avoids generic advice a model already knows
- includes freshness rules for current docs, legal, platform, model, or API claims
- validates with the official skill validator
- is mentioned in account routing when globally relevant

For bundle promotion and release review, use the
[evaluation method](evaluation/evaluation-method.md),
[skill quality rubric](evaluation/skill-quality-rubric.md), and
[security review template](evaluation/security-review-template.md).
