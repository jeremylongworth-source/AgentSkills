# Accessibility Test Plan Checklist

## Checks

- Keyboard navigation
- Focus order and visibility
- Accessible names
- Roles and states
- Screen reader flow
- Color contrast
- Error messages
- Form labels
- Reduced motion
- Captions/transcripts
- Responsive zoom
- Target size

## Evidence

- Automated scan
- Keyboard walkthrough
- Screen reader notes
- Screenshots
- Browser/device
- Severity
- Owner

## Release Gate

- Block: core task impossible.
- Warn: workaround exists but user impact remains.
- Pass: no known blocker for target scope.
