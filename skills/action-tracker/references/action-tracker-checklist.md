# Action Tracker Checklist

## Extract

- Explicit action items
- Implied follow-ups
- Decisions made
- Decisions still needed
- Blockers
- Dependencies
- Risks requiring owner action

## Action Fields

- Action
- Owner
- Due date
- Priority
- Status
- Dependency
- Evidence of completion
- Follow-up cadence

## Quality Checks

- Every action has an owner or is labeled ownerless.
- Decisions and actions are not mixed.
- High-impact blocked items are visible.
- Commitments requiring approval are marked provisional.
