# Affiliate Performance Report Checklist

## Metrics

- Clicks
- Conversions
- Conversion rate
- Revenue or commission
- Average order value
- Refunds or cancellations
- EPC when supplied
- Top products
- Top links or codes

## Attribution Caveats

- Last-click bias
- Code sharing
- Missing UTMs
- Cookie or attribution window
- Platform attribution differences
- Offline influence
- Delayed conversions

## Review

- Affiliate disclosure included
- Financial claims reviewed
- No invented metrics
- Tests are trackable
