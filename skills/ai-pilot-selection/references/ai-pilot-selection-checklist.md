# AI Pilot Selection Checklist

## Pilot Definition

- Business decision
- Hypothesis
- Target users
- Workflow
- Data and tools
- Baseline
- Success metric
- Guardrail metrics
- Owner
- Timeline

## Decision Rules

- Continue
- Revise
- Scale
- Stop

## Readiness Checks

- Data access approved
- User group identified
- Governance review complete
- Support path defined
- Rollback or stop plan clear
