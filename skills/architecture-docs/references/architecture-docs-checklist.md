# Architecture Docs Checklist

## Architecture Overview

- Scope
- Audience
- Current versus planned state
- Components
- Responsibilities
- Dependencies
- Data flow
- Control flow
- Operational assumptions
- Risks
- Open questions

## ADR Template

- Title
- Status
- Context
- Decision
- Options considered
- Consequences
- Rollback or revisit trigger
- References

## Review Checks

- Grounded in source files
- Inferences labeled
- Security-sensitive details excluded or redacted
- Diagrams optional but helpful when relationships are complex
