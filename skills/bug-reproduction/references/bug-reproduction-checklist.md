# Bug Reproduction Checklist

## Capture

- Observed behavior
- Expected behavior
- User impact
- Environment
- Version/commit
- Input data
- Frequency
- Recent changes
- Logs/screenshots

## Reduce

- Smallest steps
- Minimal fixture
- Deterministic case
- Flaky conditions
- Feature flags
- Browser/device if relevant

## Convert

- Unit regression
- Integration regression
- E2E regression
- Manual verification
