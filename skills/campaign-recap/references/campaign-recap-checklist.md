# Campaign Recap Checklist

## Required Inputs

- Sponsor objective
- Deliverables
- Live links
- Reporting date range
- Platform metrics
- Affiliate or conversion data if available
- Screenshots or source exports

## Report Sections

- Summary
- Delivered assets
- Metrics
- Audience response
- Learnings
- Caveats
- Recommendations
- Renewal angle

## Review

- Metrics are sourced
- Causality is not overstated
- Missing data is visible
- Sponsor send approved
