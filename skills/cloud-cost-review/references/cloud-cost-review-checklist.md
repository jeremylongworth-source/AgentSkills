# Cloud Cost Review Checklist

## Cost Drivers

- Compute
- Containers
- Serverless invocations
- Database size and I/O
- Storage
- Bandwidth
- Logs
- Traces
- Backups
- CI minutes
- Queues/events
- Preview environments

## Guardrails

- Budgets
- Alerts
- Quotas
- Retention limits
- Autoscaling caps
- Environment cleanup
- Cost owner
- Review cadence

## Confidence

- High: billing/export data supplied.
- Medium: service usage supplied but pricing not verified.
- Low: only architecture or assumptions supplied.
