# Dashboard Design Checklist

## Dashboard Inputs

- Audience
- Decision supported
- Refresh cadence
- Source systems
- Metric owners
- Data quality rules

## Layout

- Top-level KPIs
- Trend charts
- Segment breakdowns
- Funnel/cohort views
- Detail table
- Alerts or thresholds
- Annotations
- Caveats

## Review

- Every chart has a decision purpose
- Metrics are defined
- Filters are clear
- Sensitive data excluded or permissioned
- Publication owner assigned
