# Database Schema Review Checklist

## Model Review

- Domain entities
- Relationships
- Primary keys
- Foreign keys
- Unique constraints
- Nullability
- Defaults
- Timestamps
- Audit fields
- Soft/hard delete model

## Query Fit

- Read patterns
- Write patterns
- Indexes
- Pagination
- Filtering
- Aggregations
- Tenant/user boundaries

## Migration Risk

- Backfill required
- Locking or downtime
- Rollback path
- Data loss risk
- Compatibility with old code
- Test data and migration validation
