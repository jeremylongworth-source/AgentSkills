# Deployment Plan Checklist

## Preconditions

- Build passes
- Tests pass
- Migrations reviewed
- Config reviewed
- Secrets available and scoped
- Observability ready
- Rollback plan ready
- Support owner assigned
- Release window approved

## Steps

- Dry-run or staging
- Deploy artifact
- Apply config
- Run smoke tests
- Monitor metrics/logs
- Communicate status
- Decide continue/rollback

## Go/No-Go

- Required checks:
- Approver:
- Rollback trigger:
- Customer communication:
