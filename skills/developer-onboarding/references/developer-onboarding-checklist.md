# Developer Onboarding Checklist

## Setup

- Supported operating systems
- Required tools
- Install commands
- Environment variables
- Local services
- Test command
- Build command
- First successful run

## Repo Orientation

- Key folders
- Main entry points
- Test locations
- Generated files
- Contribution workflow
- Release workflow
- Troubleshooting links

## Review Checks

- Commands verified
- OS differences called out
- No secrets included
- First task is realistic
- Setup success is measurable
