# Funnel Analysis Checklist

## Funnel Definition

- Goal
- Start event
- Steps
- End event
- Grain
- Window
- Segment
- Timeout
- Re-entry
- Deduplication

## Quality Checks

- Event coverage
- Consistent names
- Timezone
- Bot/internal traffic
- Missing data
- Sample size
- Segment mix

## Output

- Conversion by step
- Drop-off points
- Segment comparison
- Caveats
- Follow-up action
