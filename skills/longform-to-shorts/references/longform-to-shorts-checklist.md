# Longform To Shorts Checklist

## Source Review

- Source type
- Rights/permissions
- Key moments
- Standalone context
- Quote accuracy
- Visual/audio quality

## Clip Fields

- Timestamp or section
- Hook
- Payoff
- Edit note
- On-screen text
- Caption
- CTA
- Platform

## Risk

- Missing context
- Copyright
- Sponsor disclosure
- Sensitive topic
- Unsupported claim
