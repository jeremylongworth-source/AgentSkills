# Media Kit Builder Checklist

## Sections

- Creator snapshot
- Audience profile
- Platform metrics
- Content pillars
- Brand categories
- Partnership formats
- Case studies or examples
- Contact path

## Proof Rules

- Mark metrics as verified, estimated, or missing
- Use only supplied sponsor logos or testimonials
- Separate audience fit from performance claims
- Note date range for metrics when available

## Review

- Voice matches creator
- No inflated claims
- Disclosure and rights caveats visible
- External sharing approved
