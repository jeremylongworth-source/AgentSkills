# Metric Definition Checklist

## Definition

- Name
- Purpose
- Owner
- Formula
- Grain
- Time window
- Source table/system
- Filters
- Exclusions
- Segments
- Freshness
- Quality checks
- Caveats

## Review Questions

- What decision uses this metric?
- Can two teams calculate it the same way?
- What changes the interpretation?
- What should not be included?
- Who approves definition changes?
