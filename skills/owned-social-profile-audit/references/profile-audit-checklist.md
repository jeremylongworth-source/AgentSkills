# Owned Social Profile Audit Checklist

## Profile Fields

- Display name matches the intended brand or person.
- Handle is recognizable and consistent across channels where possible.
- Bio or intro states the primary offer in buyer language.
- Website link points to the main conversion path, not a low-priority resource.
- Location, category, contact buttons, and public details support trust.
- Pinned, featured, article, or highlight surfaces support the main CTA.

## Public Visibility

- Check logged-out public view when possible.
- Check logged-in non-owner view when public view appears incomplete.
- Open direct post, article, media, and profile links separately.
- Record when a tab appears but redirects to login or fails to load.
- Capture screenshots for profile, pinned content, media tabs, and link redirects.

## Asset Review

- Profile image remains readable in a small circular crop.
- Banner/header keeps important text away from profile-photo overlap zones.
- Text-heavy assets are checked at desktop, mobile, and thumbnail sizes.
- Export PNG for text/logo-heavy graphics and JPG for photo-heavy banners where
  the platform recommends it.

## Current Platform Notes To Verify

These facts were checked against official platform docs during May 2026 work,
but should be re-verified before exact recommendations:

- X profile: official docs list a 160-character bio, 1500 x 500 header image,
  and 400 x 400 profile image. X says profile photo, display name, or handle
  changes can temporarily remove a Premium checkmark until review.
- X Articles: official docs describe Articles as available to Premium tiers and
  supporting long-form text, links, media, formatting, and profile Articles tab
  publication according to selected audience controls.
- Facebook Page: official docs say Page profile images are circular and should
  be 320 x 320 for best quality. Page cover photos use a left-aligned full-bleed
  16:9 layout, can be cropped/resized, and Facebook recommends PNG for logo or
  text-heavy profile/cover images.
- YouTube Shorts: official docs classify square or vertical videos up to three
  minutes as Shorts for standard channels when uploaded after October 15, 2024.

## Recommendation Priorities

1. Main CTA and website link.
2. Bio clarity and buyer relevance.
3. Profile image and banner trust.
4. Pinned or featured content.
5. Proof/resource links.
6. Public visibility and verification gaps.
7. Cadence and next content.
