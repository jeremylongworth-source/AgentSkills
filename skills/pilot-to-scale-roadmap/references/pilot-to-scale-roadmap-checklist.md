# Pilot To Scale Roadmap Checklist

## Pilot Evidence

- Hypothesis
- Users
- Workflow
- Metrics
- Guardrails
- Outcomes
- Learnings
- Risks
- Feedback

## Scale Gates

- Data access approved
- Security review complete
- Privacy review complete
- Legal/compliance review complete
- Monitoring and evals defined
- Support path ready
- Training ready
- Owner signoff complete

## Decision Options

- Stop
- Revise
- Extend pilot
- Scale in phases
