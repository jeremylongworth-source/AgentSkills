# Production Readiness Review Checklist

## Readiness Areas

- Requirements complete
- Tests pass
- Observability ready
- Alerts configured
- Security reviewed
- Performance acceptable
- Data/migrations safe
- Config/secrets ready
- Deployment plan ready
- Rollback plan ready
- Support docs ready
- Owner approvals recorded

## Decision

- Ship
- Ship with conditions
- Defer
- Block

## Evidence

- Test output
- Build output
- Monitoring links
- Security review
- Runbook
- Rollback plan
- Support handoff
