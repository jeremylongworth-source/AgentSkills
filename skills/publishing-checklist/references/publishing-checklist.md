# Publishing Checklist

## Pre-Publish

- Asset final
- Caption final
- Links checked
- Thumbnail/title checked
- Metadata/tags checked
- Subtitles/captions checked
- Audio quality checked
- Rights/copyright checked
- Sponsor/affiliate disclosure checked
- AI disclosure checked if needed
- Brand safety reviewed
- Approval recorded

## Post-Publish

- Verify post live
- Monitor comments
- Pin or reply plan
- Track metrics
- Log issues
- Capture sponsor proof
- Note correction if needed

## Decision

- Publish
- Publish with changes
- Hold
- Escalate
