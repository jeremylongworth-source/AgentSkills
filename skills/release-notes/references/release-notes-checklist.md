# Release Notes Checklist

## Inputs

- Version:
- Date:
- Commit range:
- Changelog:
- Pull requests:
- Issues:
- Validation output:
- Migration notes:

## Sections

- Highlights
- Added
- Changed
- Fixed
- Removed
- Deprecated
- Security
- Docs
- Validation
- Known issues

## Checks

- User-visible changes first
- Breaking changes explicit
- Commands verified
- Counts match validation output
- Links work
- Draft/pre-release labels preserved
