# Sponsorship Package Checklist

## Package Fields

- Package name
- Sponsor objective
- Platforms
- Deliverables
- Timeline
- Review rounds
- Usage-rights assumptions
- Reporting
- Exclusions

## Common Add-Ons

- Extra short-form cutdown
- Newsletter mention
- Link-in-bio window
- Paid usage extension
- Event/live mention
- Campaign recap

## Review

- Capacity is realistic
- Pricing source is clear
- Rights are not assumed
- Performance is not guaranteed
