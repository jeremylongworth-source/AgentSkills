# SQL Analysis Plan Checklist

## Inputs

- Question
- Grain
- Tables
- Fields
- Time window
- Filters
- Segments
- Metric definitions

## Query Shape

- CTEs
- Joins
- Aggregations
- Window functions
- Deduplication
- Timezone handling
- Output columns

## Validation

- Row counts
- Null checks
- Duplicate checks
- Reconcile to source
- Known totals
- Late-arriving data
- Sensitivity/redaction
