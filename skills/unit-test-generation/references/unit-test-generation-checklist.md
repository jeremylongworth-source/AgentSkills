# Unit Test Generation Checklist

## Test Cases

- Happy path
- Boundary values
- Invalid input
- Error path
- Empty/null/missing input
- Regression case
- Permission or state edge

## Quality Checks

- Deterministic
- Fast
- Isolated
- Clear assertions
- Minimal mocking
- Existing test style followed
- Test command documented
