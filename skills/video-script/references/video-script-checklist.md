# Video Script Checklist

## Structure

- Hook
- Context
- Promise
- Beat 1
- Beat 2
- Beat 3
- Proof/story
- CTA
- Loop or ending

## Production Notes

- Visual
- B-roll
- On-screen text
- Props
- Location
- Pacing
- Cutdowns

## Review

- Claims verified
- Sponsor disclosure noted
- Creator voice preserved
- CTA aligned
- No filler
